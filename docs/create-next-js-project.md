# Create NextJS App

```
npx create-next-app@latest
```

It might ask you the following question, accept it:

```
Need to install the following packages:
create-next-app@14.2.4
Ok to proceed? (y) y
```

You'll be prompted to answer the following questions:

```
√ What is your project named? ... corp
√ Would you like to use TypeScript? ... No / Yes
√ Would you like to use ESLint? ... No / Yes
√ Would you like to use Tailwind CSS? ... No / Yes
√ Would you like to use `src/` directory? ... No / Yes
√ Would you like to use App Router? (recommended) ... No / Yes
√ Would you like to customize the default import alias (@/*)? ... No / Yes
```

That will create your NextJS app using npm, as well as install all the necessary dependencies and initialize a git repository for you:
